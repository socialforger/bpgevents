Create and manage events inside BuddyPress/BuddyBoss groups. 
Supports presential and virtual events, maps (Leaflet + OSM), participation, ICS export, shortcode, widget, and template overrides.

== Description ==

BPGE Events adds a complete event system to BuddyPress and BuddyBoss groups.

**Features:**
* Create events inside groups
* Presential and virtual event types
* Leaflet maps with OpenStreetMap (no API key required)
* Participation system with AJAX
* ICS export for calendars
* Shortcode to embed events anywhere
* Widget for upcoming events
* Template override system
* Fully translatable (includes .po and .mo files)

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the “Plugins” menu
3. Go to a BuddyPress/BuddyBoss group and create an event

== Frequently Asked Questions ==

= Does it work with BuddyBoss? =
Yes, fully compatible.

= Do I need an API key for maps? =
No. Maps use Leaflet + OpenStreetMap.
